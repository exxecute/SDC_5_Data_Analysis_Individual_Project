# Презентация темы

## 1. Название проекта
**Analysis of Global Natural Resources: Consumption, Extraction, and Depletion Forecasting**

---

## 2. Цель проекта
- Исследовать текущие запасы и скорость добычи природных ресурсов (нефть, газ, уголь, вода, еда, энергия в целом).  
- Определить динамику потребления ресурсов по странам и регионам.  
- Прогнозировать время возможного истощения ресурсов при текущих темпах потребления.  
- Провести визуальный и статистический анализ данных, включая PCA и кластеризацию.  
- Предоставить рекомендации по рациональному использованию ресурсов.

---

## 3. Обоснование выбора темы
- **Актуальность:** Проблема истощения природных ресурсов и изменение климата являются глобальными вызовами.  
- **Доступность данных:** World Bank, BP Statistical Review, FAO, Our World in Data предоставляют качественные публичные данные.  
- **Методы анализа:** Проект позволяет применить EDA, визуализацию, статистические тесты, PCA и кластеризацию.  
- **Образовательная ценность:** Демонстрация навыков работы с Python и R, аналитического и статистического мышления.

---

## 4. Методы и инструменты

| Этап проекта | Инструмент | Методы |
|-------------|------------|--------|
| Предварительный анализ | Python (pandas, numpy) | Descriptive statistics, проверка данных, работа с пропусками |
| Визуализация | Python (matplotlib, seaborn) | Гистограммы, boxplot, violin plot, scatter, catplot |
| Проверка гипотез | R (stats, t.test, ANOVA, chisq.test) | t-test, ANOVA, χ², сравнение регионов |
| Многомерный анализ | Python (scikit-learn) | PCA, KMeans, кластеризация, визуализация главных компонент |
| Прогноз истощения | Python / R | Экстраполяция потребления, расчет “Years to depletion” |

---

## 5. Ожидаемые результаты
- Таблицы и графики распределения запасов и потребления ресурсов по странам и регионам.  
- Визуализация трендов потребления за последние годы.  
- Определение стран с высоким и низким уровнем потребления и ресурсной безопасности.  
- Кластеры стран с похожим профилем ресурсов.  
- Прогноз времени, когда основные ресурсы могут быть исчерпаны при текущих темпах потребления.  
- Рекомендации по рациональному использованию ресурсов.

## 7. Преимущества проекта
- **Междисциплинарность:** объединяет энергетику, экологию и экономику.  
- **Комплексность:** включает EDA, визуализацию, статистику, PCA, кластеризацию, прогнозирование.  
- **Научная и практическая значимость:** выявление угроз истощения ресурсов и анализ потребления по странам.  
- **Использование Python + R:** демонстрирует умение комбинировать инструменты для максимальной эффективности.

## 8. Следующие шаги
